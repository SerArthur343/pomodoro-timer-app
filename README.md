# Pomodoro Timer App

A simple and clean Pomodoro Timer application built with **HTML, CSS, and JavaScript**.
It helps you stay focused and productive using the Pomodoro technique.

---

## Features

* Start / Pause timer
* 25-minute focus sessions
* Break timer
* Simple and user-friendly UI

---

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Start focusing 🚀

No installation required.
